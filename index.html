Since we can't download the files embedded directly from the Moodle servers, this script will replace the https://moodle.carleton.edu domain with a domain of your choosing.
If there is an external folder of audio files, put it in a web-accessible location (such as your localhost folder) and enter that domain below.
Make sure the folder name is the full name of the language in all lowercase (e.g. japanese).
Otherwise, leave this blank.
